Poly Tower Part3. (2021/3/25 - by Fatty War, id3644@gmail.com)

This asset pack contains trap-type crusher and mines.
Each trap has 4~6 step prefabs applied.

Crusher
Look out! When you approach the crusher, it will turn into powder.

Mine
There are occasional misfires, so choose carefully and use them :)

Features:
- Each trap has 4~6 step. (Small, Medium, Large, XL ~)
- 3 different Body colours (Blue, Red, yellow)
- All traps used only one material & texture. Each tower has its own UV for each color.

Prefabs/Anim (\Assets\PolyTowerPart1\Prefabs\Anim)
- Contains animation prefabs. If you don't need animation, use the prefab in the parent folder.
